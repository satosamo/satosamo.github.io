# Propositional logic

Also called **zeroth-order logic**.  To keep the metalanguage of the definition as plain as possible we define **propositional logic** as:

- when we have:

## Something to Write

We define **symbols**.

- an alphabet $\Sigma=\mathcal{V}\cup\{\lnot,\land, (, )\}$ where $\mathcal{V}= \{p_1, p_2, \dots\}$ is the set of symbols called propositional variables

## Rules How to Write

We define **syntax**.

- a language $\mathcal{L}\subseteq \Sigma^*$ on $\Sigma$ generated by a [[formal-grammar|formal grammar]] (in BNF)
$$
\Phi ::= p_1, p_2, \dots \vert\ ( \lnot \Phi ) \ \vert \ ( \Phi \land \Psi )
$$
we may construct all others $(\lor, \implies, \iff)$ from these.
- or equivalently a language $\mathcal{L}\subseteq \Sigma^*$ satisfying:

$$
\frac{}{p \in \mathcal{L}} \ (p \in \mathcal{V}) \quad 
\frac{\varphi \in \mathcal{L}}{(\lnot \varphi) \in \mathcal{L}}  \quad
\frac{\varphi \in \mathcal{L}\ \psi \in \mathcal{L}}{(\varphi \land \psi) \in \mathcal{L}}
$$

The language $\mathcal{L}$ is called the language of all **well-formed formulas**.

## Meaning of What We Wrote

We define **semantics**.

- we have two-element set containing **True** and **False**
- a **valuation** or **truth assignment** is any function:
$$
v:\mathcal{V} \to \{0,1\}
$$
- extending truth function to well-formed formulas:
$$
l: \mathcal{L} \to \{0,1\}
$$
by recursion:
$$
l(\varphi) = 
\begin{cases}
  v(\varphi) & \text{if } \varphi \text{ is } p\in\mathcal{V} \\
  1-l(\psi) & \text{if } \varphi \text{ is } (\lnot \psi) \\
  \min(l(\psi),\ l(\chi)) & \text{if } \varphi \text{ is } ( \psi\land\chi) \\
  \max(l(\psi),\ l(\chi))  & \text{if } \varphi \text{ is } ( \psi\lor\chi) \\
  \max(1-l(\psi),\ l(\chi) & \text{if } \varphi \text{ is } (\psi\implies\chi) \\
  1\text{ iff } l(\psi)=l(\varphi)\text{ else }0& \text{if } \varphi \text{ is } (\psi\iff\chi) 
\end{cases}
$$

### 4. Semantic Notions

Let $\Gamma \subseteq \mathcal{L}$, $\varphi \in \mathcal{L}$.

| Notion | Definition |
|---|---|
| **$v$ satisfies $\varphi$** | $\bar{v}(\varphi) = 1$ |
| **$v$ satisfies $\Gamma$** | $\forall\,\gamma \in \Gamma,\; \bar{v}(\gamma) = 1$ |
| **Tautology** $\models \varphi$ | $\forall\, v : \mathcal{V} \to \{0,1\},\; \bar{v}(\varphi) = 1$ |
| **Contradiction** | $\forall\, v,\; \bar{v}(\varphi) = 0$ |
| **Satisfiable** | $\exists\, v,\; \bar{v}(\varphi) = 1$ |
| **Semantic consequence** $\Gamma \models \varphi$ | Every $v$ satisfying $\Gamma$ also satisfies $\varphi$ |
| **Logical equivalence** $\varphi \equiv \psi$ | $\Gamma = \emptyset,\; \{\varphi\} \models \psi$ and $\{\psi\} \models \varphi$ |

### 5. Proof System — Hilbert-Style Calculus

#### 5.1 Axiom Schemas (over $\{\neg, \to\}$)

$$
\begin{align}
\mathbf{A1}&: (\varphi \to (\psi \to \varphi)) \\
\mathbf{A2}&: ((\varphi \to (\psi \to \chi)) \to ((\varphi \to \psi) \to (\varphi \to \chi))) \\
\mathbf{A3}&: ((\neg\,\psi \to \neg\,\varphi) \to (\varphi \to \psi))
\end{align}
$$

Any substitution instance of A1–A3 is an **axiom**.

#### 5.2 Rule of Inference

**Modus Ponens (MP):**

$$\frac{\varphi \qquad (\varphi \to \psi)}{\psi}$$

#### 5.3 Proof and Derivability

A **proof** of $\varphi$ from $\Gamma$ is a finite sequence $\varphi_1, \varphi_2, \ldots, \varphi_n$ where $\varphi_n = \varphi$ and each $\varphi_i$ is either:
- An axiom, or
- A member of $\Gamma$, or
- Obtained from earlier $\varphi_j, \varphi_k$ $(j,k < i)$ by MP.

Write $\Gamma \vdash \varphi$ ("$\varphi$ is syntactically derivable from $\Gamma$").

---

### 6. Fundamental Metatheorems

| Theorem | Statement |
|---|---|
| **Soundness** | $\Gamma \vdash \varphi \;\Rightarrow\; \Gamma \models \varphi$ |
| **Completeness** (Gödel/Post) | $\Gamma \models \varphi \;\Rightarrow\; \Gamma \vdash \varphi$ |
| **Compactness** | $\Gamma \models \varphi \;\Rightarrow\; \exists\,\Gamma_0 \subseteq_{\text{fin}} \Gamma,\; \Gamma_0 \models \varphi$ |
| **Deduction Theorem** | $\Gamma \cup \{\varphi\} \vdash \psi \;\Leftrightarrow\; \Gamma \vdash (\varphi \to \psi)$ |
| **Decidability** | There exists an algorithm deciding $\models \varphi$ for any $\varphi$ (truth tables; PSPACE-complete in general for $\models$; co-NP-complete for $\not\models$) |

Together, **Soundness + Completeness** give:

$$\boxed{\Gamma \vdash \varphi \;\iff\; \Gamma \models \varphi}$$

meaning syntactic provability and semantic truth-in-all-models **coincide** exactly.

---

### Summary Structure

$$\text{Propositional Logic} = \underbrace{(\Sigma,\, \mathcal{L})}_{\text{syntax}} + \underbrace{(\{0,1\},\, \bar{v})}_{\text{semantics}} + \underbrace{(\text{Axioms},\, \text{MP})}_{\text{proof system}}$$

with the guarantee that these three components are **perfectly aligned** via the Soundness and Completeness theorems.
